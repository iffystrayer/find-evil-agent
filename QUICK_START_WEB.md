# 🚀 Quick Start - Web Interface

## Launch in 3 Steps

```bash
# 1. Activate environment
cd /Users/ifiokmoses/code/find-evil-agent
source .venv/bin/activate

# 2. Launch web interface
find-evil web

# 3. Open browser
# http://localhost:17000
```

## Try These Examples

### Example 1: Ransomware Analysis (30 seconds)

**Single Analysis Tab:**
- **Incident:** Ransomware detected encrypting files on Windows server
- **Goal:** Identify malicious processes and IOCs
- **Format:** HTML
- Click **Analyze**

**Expected:** Professional report with MITRE ATT&CK mapping, IOC tables, interactive graph
